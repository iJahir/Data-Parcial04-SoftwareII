# Carpeta de Datos (`/data`)

## Propósito
El propósito de esta carpeta es almacenar todos los conjuntos de datos (datasets) en su formato original y procesado que se utilizarán durante el análisis del proyecto. 

## Archivos que deben guardarse aquí
- Archivos `.csv`, `.excel`, `.json` o cualquier otro formato tabular proporcionado.
- Datasets originales proporcionados por el docente (sin modificar).
- Datasets limpios o procesados listos para análisis y modelado.

## Ejemplos de contenido
- `clave_C_agrupacion.csv`
- `clave_C_asociacion.csv`
- `clave_C_correlacion.csv`
