# Carpeta de Informe (`/informe`)

## Propósito
El propósito de esta carpeta es contener toda la documentación escrita, incluyendo borradores y versiones finales del informe de resultados del proyecto.

## Archivos que deben guardarse aquí
- Documentos de texto (`.docx`, `.pdf`) con el reporte estructurado del análisis.
- Presentaciones del proyecto, si aplica.
