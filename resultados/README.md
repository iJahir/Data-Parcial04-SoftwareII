# Carpeta de Resultados (`/resultados`)

## Propósito
El propósito de esta carpeta es guardar las salidas generadas por los modelos y análisis, como modelos entrenados, tablas de resumen, o listas de reglas de asociación extraídas.

## Archivos que deben guardarse aquí
- Archivos exportados desde el notebook con resultados (.csv, .xlsx).
- Modelos serializados (ej. archivos `.pkl` o `.joblib`).
- Reportes automatizados generados por código (ej. perfiles de datos en HTML).
