# Carpeta de Notebooks (`/notebook`)

## Propósito
El propósito de esta carpeta es albergar los Jupyter Notebooks o scripts de Google Colab donde se desarrolla y documenta el código del proyecto, incluyendo carga de datos, análisis exploratorio (EDA), y modelos de Machine Learning (Clustering K-Means).

## Archivos que deben guardarse aquí
- Archivos de extensión `.ipynb` (Jupyter/Colab).
- Scripts auxiliares de Python `.py` (si se modulariza el código).

## Ejemplos de contenido
- `Analisis_Exploratorio_y_Clustering.ipynb`
- `1_EDA_Dataset_Docente.ipynb`
- `2_Modelo_KMeans_y_Reglas.ipynb`
